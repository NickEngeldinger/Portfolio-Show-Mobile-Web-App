Portfolio Show
Directory

Welcome to the Art Institute of Portland Summer 2011 portfolio show directory mobile web app. Through this app you can access the complete roster of graduates in the Summer 2011 class. You can select a specific program from the departments page and view an alphabetical list of graduates from that program. If a graduate's name has a icon next to it you can press their name to view a short bio and a link to their digital portfolio or email address.
